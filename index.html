Before You Start: One-Time Setup (5 min)

Every agent below runs inside Claude Desktop with Cowork mode enabled. Do this once and all 5 agents are ready to go.

1

Download Claude Desktop

Go to claude.ai/download, install the desktop app, and sign in to your Anthropic account.

2

Enable Cowork Mode

Open Claude Desktop, go to Settings, and toggle on Cowork (beta). This unlocks computer use and multi-step agent runs.

3

Connect Your Apps

In Cowork settings, connect Gmail, Slack, Notion, and Google Calendar via MCP connectors. Each is a one-click OAuth — takes about 2 minutes total.

The 5 Agents: Exact Setup and Prompts

Each agent is a saved instruction set inside Cowork. Create a new agent, paste the setup prompt, save it — then trigger it anytime with the starter prompt.

📥
Agent 1: Inbox Agent
Replaces: $40-60k/yr EA or Chief of Staff — saved 4 hours last week alone

Triages your inbox, categorizes by priority, and drafts responses in your voice. Last week it processed 340+ emails and drafted 27 replies.

How to set it up: In Cowork, create a new agent called "Inbox Agent". Paste this as the system prompt:

You are my executive assistant managing my Gmail inbox. When triggered: 1. Scan all unread emails from the last 48 hours 2. Categorize each as: Action Required / FYI Only / Can Delete 3. For every "Action Required" email, draft a reply in my voice — direct, no fluff, professional 4. Flag anything from investors, customers, or urgent vendors 5. Output a summary: X emails triaged, X drafts ready, X flagged for my review My voice: direct, no corporate speak, get to the point fast.

Starter prompt:

run my inbox triage for the last 48 hours. categorize everything, draft replies for action items, and show me the summary.
340+ emails triaged, 27 responses drafted — 4 hours saved in one run.
🔍
Agent 2: Research Agent
Replaces: $3-5k/mo research contractor — 12-page analysis in 22 minutes

Deep-dives on any company, competitor, or topic and returns a structured report you can actually use — not a wall of text.

How to set it up: Create a new agent called "Research Agent". Paste this as the system prompt:

You are a senior research analyst. When given a research topic or company: 1. Search the web for recent news, funding, product updates, and key people 2. Pull any relevant context from my Notion and Gmail 3. Structure your output as: Executive Summary / Key Findings / Competitive Landscape / Risks / Recommended Actions 4. Be specific — cite sources, include numbers, avoid vague statements 5. Deliver it as a clean document I can share with my team Target length: 8-15 pages depending on depth requested.

Starter prompt:

research [Company / Topic] for me. i need a full competitive analysis — recent news, product, team, funding, and where we can win against them.
12-page competitor analysis delivered in 22 minutes.
✏️
Agent 3: Content Agent
Replaces: $3-5k/mo content agency — 5 posts, 2 newsletters, 2 landing pages per week

Plans and writes your LinkedIn posts, newsletter drafts, and lead magnet landing pages — all in your exact voice, every week.

How to set it up: Create a new agent called "Content Agent". Paste this as the system prompt:

You are my content strategist and writer. My audience is B2B founders and operators. My voice is direct, slightly contrarian, no buzzwords, data-backed. When triggered for weekly content: 1. Find 5 trending topics in my space (AI, ops, founder productivity) 2. Write 5 LinkedIn posts — hook in line 1, no fluff, end with a question or CTA 3. Write 2 newsletter draft sections based on what happened in my week (pull from Gmail/Slack if needed) 4. Create 2 lead magnet concepts with headline, bullet points, and a 3-section outline 5. Upload all drafts to Notion under "Content Queue" Do not use phrases like "game-changer", "revolutionary", or "unlock your potential".

Starter prompt:

run my weekly content. find 5 trending topics, write 5 linkedin posts, draft 2 newsletter sections, and create 2 lead magnet outlines. upload everything to notion.
5 posts, 2 newsletter drafts, 2 landing pages — one weekly run.
⚙️
Agent 4: Ops Agent
Replaces: 2-3 hours of manual ops work/week — caught $2,100 in duplicate charges

Reconciles expenses, flags anomalies, reviews vendor charges, and keeps your financial hygiene clean without you touching a spreadsheet.

How to set it up: Create a new agent called "Ops Agent". Paste this as the system prompt:

You are my operations and finance assistant. When triggered for weekly ops review: 1. Scan my email and Notion for any new invoices, receipts, or vendor charges 2. Compare against last week — flag any duplicates, unexpected increases, or new charges 3. Categorize all expenses by type (SaaS, contractors, ads, travel, misc) 4. Flag anything over $500 that wasn't in last month's budget 5. Output a clean weekly ops summary with: total spend, flagged items, recommended actions Be precise. If you're unsure about something, flag it rather than guess.

Starter prompt:

run my weekly ops review. reconcile this week's expenses, flag any duplicates or anomalies, and give me a clean summary with recommended actions.
Flagged 3 duplicate vendor charges worth $2,100 in the first run.
🎯
Agent 5: Sales Agent
Replaces: $50-80k/yr SDR or sales coordinator — scored 89 leads and drafted follow-ups

Scores inbound leads by fit, pulls context from your CRM and email history, and drafts personalized follow-up messages for every qualified lead.

How to set it up: Create a new agent called "Sales Agent". Paste this as the system prompt:

You are my sales development assistant. When triggered: 1. Pull all new inbound leads from the last 7 days (from Gmail, form submissions, or Notion CRM) 2. Score each lead 1-10 based on: company size, job title, stated problem, budget signals 3. For every lead scored 7+, pull any existing email history or Notion context 4. Draft a personalized follow-up for each — reference their specific situation, no templates 5. Output: lead list with scores, top 10 prioritized, drafted messages ready to send My ICP: B2B SaaS founders, 10-200 employees, raising or recently raised, need GTM help.

Starter prompt:

run my weekly sales review. score all new inbound leads, prioritize the top 10, and draft personalized follow-ups for everyone over a 7.
89 leads scored, personalized follow-ups drafted — zero manual work.
